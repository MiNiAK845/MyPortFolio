README – Personal Portfolio Website
Introduction
This project is a modern, glowing, animated personal portfolio website created using HTML5, CSS3, and JavaScript. The purpose of this portfolio is to display personal information, skills, sample projects, and provide a contact section in a clean and visually appealing manner.
The website is fully responsive and includes animations, interactive JavaScript features, and a user-friendly design.

Features
1. Animated Gradient Background
The background smoothly animates with glowing colors to create a modern UI effect.
2. Floating Neon Particles
Small glowing particles float in the background to give a futuristic aesthetic.
3. Responsive Navigation Bar
•	Works on desktop & mobile
•	Includes hamburger menu for small screens
•	Highlights active page

4. Typing Animation (Home Page)
JavaScript automatically types text letter-by-letter to grab user attention.
5. Scroll Animations
Sections fade in smoothly as you scroll down.
6. Floating Project Cards
Project cards rise slightly when hovered, glowing gently.
7. Glowing Buttons & Borders
The design uses glowing borders, buttons, and section dividers for visual appeal.
8. Fully Responsive Layout
Looks great on:
•	Mobiles
•	Tablets
•	Laptops
•	Desktop monitors

How to Run the Project
1.	Download/clone the entire project folder.
2.	Ensure all HTML files, CSS, JS, and images are in correct directories.
3.	Open index.html in a browser (Chrome recommended).
4.	Navigate using the navbar.
